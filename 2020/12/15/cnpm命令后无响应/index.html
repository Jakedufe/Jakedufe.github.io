cnpm命令后无响应
更新于 2020-12-15 web 转载 热度 1 喜欢 0

问题：

  1.把前端环境配制完毕之后，打开项目，输入cnpm install之后，光标一直在另起一行的位置闪，但是丝毫没有在安装的迹象。

   2.打开cmd，输入cnpm -v 之后，和问题1一样。

   3.此时重新安装npm，输入npm install cnpm -g --registry=https://registry.npm.taobao.org命令，也没有任何反应。

解决方法：

   找到C盘路径下的用户，找到一个名为.npmrc的文件，删掉即可。
